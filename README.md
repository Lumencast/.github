# Lumencast org-wide community health files

This repo hosts the assets shown across the entire `Lumencast` GitHub organization :

- **`profile/README.md`** — landing page rendered at [github.com/Lumencast](https://github.com/Lumencast)

Future additions :

- `CODE_OF_CONDUCT.md`, `CONTRIBUTING.md`, `SECURITY.md` — fall-through community files for repos that don't ship their own
- `FUNDING.yml` — sponsorship configuration
- Issue / PR templates that apply org-wide

GitHub's special-repo conventions documented at <https://docs.github.com/en/communities/setting-up-your-project-for-healthy-contributions/creating-a-default-community-health-file>.
